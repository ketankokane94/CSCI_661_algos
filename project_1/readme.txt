Instructions to execute the code
The code is divided into 3 part 
Data set generation (generate_dataset.py)
The actual implementation of the two quicksort algorithm (qsort, qsort2)
Driver class which uses above two parts of the code to run the experiment (sort_data_experiment.py)

The system expects ‘numpy’ and ‘time’ module all ready to be installed.
and requires all the files stay in same directory when executing the below code

Code to run the experiment 
Python3 sort_data_experiment.py
